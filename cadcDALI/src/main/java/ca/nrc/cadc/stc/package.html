The CADC STC library

The CADC STC library is an implementation of STC-S , a String serialization of STC.

ca.nrc.cadc.wcs.STC

The class ca.nrc.cadc.wcs.STC contains factory methods to create a Region, SpectralInterval, or AstroCoordArea from a STC-S phrase, and to build a STC-S phrase from a Region, SpectralInterval, or AstroCoordArea.

STC-S Phrase

A STC-S phrase consists of one or more sub-phrases. Each sub-phrase pertains to a particular coordinate frame. The sub-phrases are concatenated together to form the phrase, and shall be in the following order:

 Time Space Spectrum Redshift

Currently only Space and Spectrum sub-phrases are supported.

Space Sub-phrase

The following syntax is supported to describe a Space sub-phrase.

 Box [<frame>] [<refpos>] [<flavor>] <pos> <bsize>

 Circle [<frame>] [<refpos>] [<flavor>] <pos> <radius>

 Polygon [<frame>] [<refpos>] [<flavor>] <pos> ...

 Position [<frame>] [<refpos>] [<flavor>] <pos>

 Union [<frame>] [<refpos>] [<flavor>] ( <region> ... )

 Not ( <region> )

 Intersection [<frame>] [<refpos>] [<flavor>] ( <region> ... )

Notes:

  • <frame> is one of the following: ECLIPTIC, FK4, FK5, GALACTIC, ICRS, UNKNOWNFRAME
  • <flavor> is one of the following: CARTESIAN2, CARTESIAN3, SPHERICAL2
  • <refpos> is one of the following: BARYCENTER, GEOCENTER, HELIOCENTER, LSR, TOPOCENTER, RELOCATABLE, UNKNOWNREFPOS
  • <pos> <bsize> are pairs of doubles
  • <radius> is a double
  • <region> is a Space sub-phrase, one of the following: Box, Circle, Polygon, Position, Union, Not, Intersection

Spectrum Sub-phrase

The following syntax is supported to describe a Spectrum sub-phrase.

 SpectralInterval <lolimit> <hilimit> unit <unit>

Notes:

  • <lolimit> <hilimit> are doubles
  • <unit> is one of the following: Hz, MHz, GHz, m, mm, um, nm, Angstrom, eV, keV, MeV
